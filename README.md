# WEEK-5 Project

Welcome to the **WEEK-5** project repository! This directory contains various frontend development assignments and projects, structured clearly into daily tasks. The focus is on HTML/CSS layouts, responsive design, and an introduction to React basics.

## Project Structure

The repository is modularized into three main daily folders:

### 1. `html-css-basics`
This folder focuses on basic HTML structures and CSS styling.
* **`tables-and-user-profile/`**: Demonstrates the creation of simple and complex tables (using `rowspan` and `colspan`), along with a styled user profile list.
* **`flexbox-cards/`**: An introduction to Flexbox, displaying a row of neatly styled cards with a parallax-like background effect.

### 2. `responsive-layouts`
This folder focuses on responsive web design techniques and media queries.
* **`marketing-template/`**: A classic marketing page layout featuring a large "Jumbotron" hero section and three supporting content cards.
* **`responsive-cards/`**: A responsive layout with a header, hero section, and a flexible grid of information cards that stack on smaller screens.
* **`city-guide/`**: A responsive city guide template showcasing a navigation bar, a hero banner, and a grid-based content layout.

### 3. `react-basics`
An introductory React application built using **Vite** and styled with **Tailwind CSS**.
* **Features included:**
  * Component-based architecture (`Navbar`, `Footer`, `Products`, `Counter`).
  * State management and event handling (Counter component).
  * Passing props and list rendering (Products component).
* **To run the application locally:**
  ```bash
  cd react-basics
  npm install
  npm run dev
  ```
